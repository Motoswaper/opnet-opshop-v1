# 🛒 OPSHOP v1  
### Trustless Marketplace Protocol for OP_NET

Minimal, deterministic, trust‑aware marketplace built on:

- OP20 (payments)
- TrustLayer (reputation)
- MotoSettle (escrow + disputes)

---

# 📘 Overview

OPSHOP v1 enables:

- creating listings  
- purchasing listings  
- trust‑aware interactions  
- escrow‑protected trades  

---

# 🔗 Integrations

- **TrustLayer** — validates sellers/buyers  
- **MotoSettle** — handles disputes  
- **OP20** — payment asset
